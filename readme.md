# ChatBot IA 🤖

Um chatbot com inteligência artificial construído com Python e Flask, integrado com a API do Groq (LLaMA 3.3).

## 🔗 Acesse online
[Clique aqui para testar](https://chatbotiapython.onrender.com)

## 🛠️ Tecnologias
- HTML, CSS e JavaScript
- Python + Flask
- Groq API (LLaMA 3.3 70b)

## 🚀 Como rodar localmente
1. Clone o repositório
2. Crie um arquivo `.env` com sua chave: `GROQ_KEY=sua_chave`
3. Instale as dependências: `pip install -r requirements.txt`
4. Rode o servidor: `python server.py`
5. Acesse `http://127.0.0.1:3000`
